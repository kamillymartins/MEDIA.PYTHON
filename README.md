# 📚 Calculadora de Média Escolar

## 📖 Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de calcular a média de um aluno com base em três notas informadas pelo usuário. Ao final da execução, o sistema exibe um boletim contendo o nome do aluno, a média obtida e sua situação acadêmica.

---

## 🚀 Funcionalidades

* Cadastro do nome do aluno;
* Entrada de três notas;
* Cálculo da média aritmética;
* Verificação da situação do aluno;
* Exibição de um boletim formatado.

---

## 🛠️ Tecnologias Utilizadas

## 📋 Critério de Aprovação

| Média                | Situação  |
| -------------------- | --------- |
| Maior ou igual a 7,0 | Aprovado  |
| Menor que 7,0        | Reprovado |
